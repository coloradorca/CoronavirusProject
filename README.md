# Coronavirus Project

### An intearactive map with a graphical represenatation of timeseries data

---

## Installation

Ensure `node` and `npm` are installed

Navigate to preferred local directory

Clone this repo

Run the following commands

`npm install`

`npm start`

---

## Technologies used

ArcGIS Javascript API (v 4.15)

React

Chart.js

D3
